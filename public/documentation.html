This is "MovieBase". MovieBase is an API intended to provide users with access to information about their favorite movies.

Business logic URL HTTP Method parameters Request body data format Response body data format
return a list of all movies /movies get none none a json object with all movies
return data on a single movie as per user request /movies/:title get title none a json object with data on the requsted movie title
return a description about a genre i.e Science Fiction /movies/genre/:genreName get genreName none a json object with genre description
return info on director from director name /movies/director/:directorName get directorName none a json object with director info
allow users to register /users post none a json object containing user Name eg: { Name: '' } a json object showing newUser data
allow user to change name /users/:id put id a json object with new updated name eg: { Name: '' } a json object showing updatedUser.Name
allow user to add a movie to their favoriteMovie array /users/:id/:movieTitle post id, movieTitle a json object with new movieTitle in array favoriteMovies eg: { Title: '' } a text response confirming the title has been added
allow user to delete a movie from their list /users/:id/:movieTitle delete id, movieTitle a json object with the movieTitle to be removed a text response confirming the title has been removed
allow user to unregister /users/:id delete id a json object with the user id number a text response confirming user id has been deleted
